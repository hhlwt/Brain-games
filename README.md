<h1>This is a collection of simple console games.</h1>

<h1>How it works?</h1>
<h2>You should make the following steps to try it yourself:</h2>
<ol>
  <li>git clone git@github.com:hhlwt/frontend-project-lvl1.git</li>
  <li>make install</li>
  <li>sudo npm link</li>
</ol>
